# Mini-ERP Stok ve Sipariş Yönetim Sistemi

Bu proje, küçük ve orta ölçekli ticari işletmelerin ürün yönetimi, stok takibi, tedarik ve sipariş süreçlerini yönetmelerine yardımcı olmak için tasarlanmış bir Mini-ERP sistemidir.

## Özellikler
- Ürün Yönetimi (Katalog)
- Tedarik ve Depo Girişleri
- Satış ve Sipariş Yönetimi
- Durum Paneli (Dashboard)

## Kullanıcı Rolleri
- Yönetici
- Operasyon Kullanıcısı (Satış/Tedarik)

## Teknoloji Yığını
- Web Arayüzü
- Uygulama Katmanı
- Veritabanı
